This is an educational open-source project. We welcome educators, frontend developers, and AI enthusiasts to help us build the future of EdTech.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your machine.
3.  **Create a branch** for your fix or feature:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Commit your changes** with clear, descriptive messages.
5.  **Push to the branch**:
    ```bash
    git push origin feature/amazing-feature
    ```
6.  **Open a Pull Request** and link it to an existing Issue.

## Issue-Driven Development

We follow an **Issue-Driven Development** process. This means:
-   **No code without an Issue.** Every Pull Request must be linked to an open Issue.
-   **Discuss first.** If you have a major idea, open an Issue or start a Discussion before writing code.

## Pull Request Guidelines

-   **Title**: Use a clear title (e.g., `Add Veo 3 video integration`).
-   **Description**: Explain *what* changed and *why*.
-   **Screenshots**: If you changed the UI, please include screenshots or a video.
-   **Tests**: Ensure your changes don't break existing functionality.

## Code of Conduct

Please be respectful and professional in all interactions. We are here to learn and build together.
